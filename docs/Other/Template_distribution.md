# テンプレート配布

## この文書について
この文書は、開発者により作成されたAnygetのテンプレートを配布するページです。テンプレートは随時追加されます。

テンプレートは、スタートメニューまたはメインパネルの「セーブ/ロード」モードからロードすることが可能です。

バージョンの違いにより、配布されるテンプレートの互換性が保てない場合があります。ご了承ください。
